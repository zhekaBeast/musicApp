## Music App

Простое Android‑приложение для поиска и прослушивания музыки, созданное с использованием **Kotlin**, **Jetpack Compose** и архитектуры **MVVM**.

## Возможности

- **Поиск треков**: поиск музыки по названию и исполнителю.
- **Избранное**: сохранение треков в избранное и управление ими.
- **Плейлисты**: создание, редактирование и удаление плейлистов, добавление треков в плейлисты.
- **Детальный экран трека**: подробная информация о треке.

## Технологии

- **Язык**: Kotlin
- **UI**: Jetpack Compose
- **Архитектура**: MVVM
- **DI**: Koin
- **БД**: Room
- **Навигация**: Android Navigation

## Требования

- Android Studio 2025.3.1 или новее
- JDK 11 или выше
- Min SDK: 24
- Target SDK: 36

## Запуск проекта

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/USERNAME/musicApp.git
   ```
2. Откройте папку `musicApp` в Android Studio.
3. Дождитесь синхронизации Gradle.
4. Запустите приложение на эмуляторе или подключённом устройстве.

## Структура проекта

- `app/src/main/java/com/example/musicapp/ui` — экраны и UI‑компоненты.
- `app/src/main/java/com/example/musicapp/domain` — доменные модели и репозитории.
- `app/src/main/java/com/example/musicapp/data` — работа с сетью и базой данных (Room).
